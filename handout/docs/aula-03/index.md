# Aula 3 — Otimização

Nesta aula, o foco é entender o **Gradiente Descendente** como algoritmo de otimização.

A ideia não é tratar a Regressão Logística como tema principal, mas usar esse modelo como exemplo para enxergar como os parâmetros são ajustados iterativamente ao longo do treinamento.

A teoria da aula está concentrada na próxima seção, [Otimização: Gradiente Descendente](gradient-descent.md), e a aplicação prática acontece no notebook da aula.
