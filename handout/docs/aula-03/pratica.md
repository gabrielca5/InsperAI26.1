# Prática

O notebook `otimizacao.ipynb` é a atividade principal da aula 3.
Nele, você implementa uma versão manual de regressão logística com **batch gradient descent**.

O ponto principal, porém, não é “fazer outro classificador”.
É usar esse exemplo para enxergar o mecanismo de otimização funcionando passo a passo.

## O Que Você Vai Fazer

1. Implementar o cálculo do gradiente.
2. Implementar o algoritmo de gradiente descendente.
3. Aplicar o gradiente descendente para treinar os parâmetros.
4. Comparar os parâmetros obtidos com os do `scikit-learn`.
5. Avaliar o modelo treinado.
