# Preparando o Ambiente

Antes de escrever qualquer linha de código, precisamos montar a estrutura que vai sustentar todo o trabalho ao longo do semestre.

Nesta seção você vai instalar e configurar quatro ferramentas essenciais:

- **uv** — gerenciador de pacotes e ambientes virtuais Python
- **Git** — controle de versão local
- **GitHub** — onde o código vive na nuvem
- **VS Code** — o editor que vamos usar no dia a dia

Não é necessário entender tudo de uma vez. O objetivo aqui é ter o ambiente funcionando, e o significado de cada peça vai ficando mais claro conforme o avançamos.
